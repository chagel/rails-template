## chagel/rails-template 

This is a Rails 4.2 boilerplate for new project bootstrap.

### Gems featured

- Default database: Mysql
- Rspec and Guard 
- Sidekiq 
- Capistrano 

### How to install

`rails new PROJECT -m https://raw.githubusercontent.com/chagel/rails-template/master/template.rb`
